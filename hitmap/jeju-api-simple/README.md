# 제주 관광 API 간단 구현

제주관광빅데이터(data.ijto.or.kr)의 주요 기능을 카카오 지도 API를 활용하여 재구현한 프로젝트입니다.

## 주요 기능

### 1. 카카오 지도 API
- 제주도 중심의 인터랙티브 지도
- 지도 타입 변경 (일반/스카이뷰)
- 줌 컨트롤

### 2. 키워드 검색
- 사용자가 입력한 키워드로 장소 검색
- 제주도 반경 50km 이내 검색
- 검색 결과를 지도와 리스트로 표시

### 3. 카테고리별 검색
- **관광지** (AT4): 제주 주요 관광 명소
- **음식점** (FD6): 맛집, 식당
- **카페** (CE7): 카페, 디저트
- **숙박** (AD5): 호텔, 펜션, 게스트하우스
- **편의점** (CS2): 편의점
- **약국** (PM9): 약국
- **병원** (HP8): 병원, 의료기관
- **주유소** (OL7): 주유소

### 4. 날씨 정보
- 헤더에 현재 제주 날씨 표시 (시뮬레이션)
- 실제 구현시 OpenWeatherMap API 등과 연동 가능

### 5. 검색 결과
- 장소명, 주소, 전화번호, 거리 정보 표시
- 클릭 시 지도 중심 이동 및 정보창 표시
- 마커 클릭으로 상세 정보 확인

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 반응형 디자인, 그라디언트, 애니메이션
- **Vanilla JavaScript**: ES6+ 문법
- **Kakao Maps API**: 지도, 장소 검색, 카테고리 검색

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 카테고리 버튼 클릭 또는 검색어 입력
3. 지도에서 마커 클릭하거나 오른쪽 리스트에서 장소 선택
4. 상세 정보 확인

## 반응형 디자인

- 데스크톱: 지도 + 사이드 패널
- 태블릿: 지도 + 하단 패널
- 모바일: 세로 스택 레이아웃

## 개선 가능 사항

1. **실시간 날씨 API 연동**: OpenWeatherMap, 기상청 API
2. **길찾기 기능**: 카카오 내비게이션 연동
3. **즐겨찾기**: localStorage를 활용한 저장 기능
4. **실시간 교통정보**: 카카오 교통정보 API
5. **리뷰/평점**: 사용자 리뷰 시스템
6. **다국어 지원**: i18n 라이브러리 활용
